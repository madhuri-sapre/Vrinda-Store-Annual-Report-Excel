# ECOMMERCE_SALES_ANALYSIS_DASHBOARD_EXCEL

Description:
Developed an interactive Excel dashboard to analyze and visualize e-commerce sales performance across multiple platforms and customer segments. The dashboard provides key insights into total sales, order status, customer demographics, and regional performance, enabling data-driven business decisions.

Key Features:
Created dynamic charts to analyze monthly sales trends and order volumes.
Designed interactive slicers for filtering by month, channel, and product category.
Built visual comparisons for gender-wise and age-group sales distribution.
Highlighted top-performing states and sales contribution by e-commerce channels such as Amazon, Flipkart, Meesho, and Myntra.
Displayed order status breakdown (Delivered, Returned, Refunded, Cancelled) to track performance efficiency.

Tools & Skills Used:
Microsoft Excel, Pivot Tables, Charts, Data Cleaning, Dashboard Design, Data Visualization, Business Analysis.

Outcome:
Delivered a comprehensive, easy-to-use dashboard that helps stakeholders monitor e-commerce performance, understand customer behavior, and identify high-performing regions and channels to improve sales strategy.
